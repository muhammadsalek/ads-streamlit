# 🎓 Academic Design Studio (ADS)

## 🚀 Features

- 📝 Assignment Builder
- 📊 Lab Report Builder
- 🎨 Poster Studio
- 📄 Academic CV Builder
- 📱 QR Code Generator
- 📦 Export Manager
